## English

### What is Nano Banana PPTX?

Nano Banana PPTX transforms your ideas into **stunning, AI-generated presentations** where every slide is a bespoke full-bleed visual powered by **Google's Gemini 3 Pro Image model**. No more generic templates — each presentation is a unique work of art assembled into a professional `.pptx` file.

```
Topic → Outline → Slide Config (JSON) → AI Images → PPTX
```

### Key Features

- **Pixel-perfect AI design** — Every slide is a unique, bespoke visual generated by Gemini 3 Pro
- **Text rendered in-image** — Crisp titles, bullets, and statistics rendered directly in the AI image
- **5 world-class design styles** — Midnight Editorial, Warm Editorial, Kinetic Dark, Luxe Minimal, Bold Geometric
- **Style consistency** — Slide 1 image is passed as reference to all subsequent slides for visual coherence
- **4K resolution** — Razor-sharp on any projector or screen
- **Flexible placeholder system** — `{{TOKEN}}` syntax lets you swap content without modifying prompts
- **JSON-driven workflow** — Full control, versionable configs, easy to regenerate individual slides

### Quick Start

#### 1. Install Dependencies

```bash
npm install -g @google/generative-ai pptxgenjs
```

#### 2. Set Environment Variable

```bash
export GEMINI_API_KEY="your-google-ai-studio-key"
```

Get your free API key at [aistudio.google.com](https://aistudio.google.com)

#### 3. Generate Your First Presentation

```bash
node scripts/quick_start.js \
  --topic  "The Future of Sustainable Architecture" \
  --style  A \
  --slides 10 \
  --output deck.pptx
```

**Available Styles:**
| Key | Name | Inspiration |
|-----|------|-------------|
| `A` | Midnight Editorial | Bloomberg Businessweek, Wallpaper |
| `B` | Warm Editorial | Kinfolk, Dezeen |
| `C` | Kinetic Dark | Apple WWDC, Google I/O |
| `D` | Luxe Minimal | McKinsey, LVMH |
| `E` | Bold Geometric | Pentagram, Sagmeister & Walsh |

### Usage Workflow

1. **Generate outline & config:**
   ```bash
   node scripts/quick_start.js --topic "Your Topic" --style A --slides 10
   ```
   This creates a `slides-<slug>.json` config file (editable).

2. **Edit the config** — customize text, add/remove slides, adjust placeholders.

3. **Generate the PPTX:**
   ```bash
   node scripts/generate.js --config slides-<slug>.json --output deck.pptx
   ```

4. **QA check** — inspect generated images in `./slides/` folder.

5. **Re-generate failed slides** — edit the prompt in JSON, re-run `generate.js`.

### Configuration

#### Environment Variables

| Variable | Required | Default | Description |
|----------|----------|---------|-------------|
| `GEMINI_API_KEY` | Yes | — | Google AI Studio API key |
| `GEMINI_MODEL` | No | `gemini-3-pro-image-preview` | Override image model |

#### Placeholder System

Use `{{TOKEN}}` syntax in slide configs. Global variables are available on every slide:

| Token | Description |
|-------|-------------|
| `{{PAGE}}` | Current slide number (auto) |
| `{{TOTAL}}` | Total slide count (auto) |
| `{{TITLE}}` | Presentation title |
| `{{SUBTITLE}}` | Subtitle |
| `{{STYLE_A_SYSTEM}}` | Style A design system prompt |
| `{{SECTION}}` | Section label |
| `{{BULLETS}}` | Bullet point list |
| `{{IMAGE_DESCRIPTION}}` | AI image subject description |
| `{{STATS}}` | Statistics (format: `VALUE \| LABEL \| CONTEXT`) |
| `{{QUOTE_TEXT}}` | Quote content |
| `{{SPEAKER_NAME}}` | Quote attribution |

### Troubleshooting

| Issue | Fix |
|-------|-----|
| Text garbled or wrong | Write text verbatim in prompt: `Title: "exact text"` |
| Style inconsistency | Ensure slide 1 succeeds first — it's the reference for all others |
| Rate limit error | Increase `throttle_ms` in config (try `4000`) |
| Blank/no image in response | Retry — generator retries failed slides automatically |
| API key error | Verify `GEMINI_API_KEY` env var at [aistudio.google.com](https://aistudio.google.com) |

### 什么是 Nano Banana PPTX？

### 配置

#### 环境变量

| 变量 | 必填 | 默认值 | 描述 |

在幻灯片配置中使用 `{{TOKEN}}` 语法。全局变量在每张幻灯片中可用：

| `{{TITLE}}` | 演示文稿标题 |

| `{{QUOTE_TEXT}}` | 引用内容 |
| `{{SPEAKER_NAME}}` | 引用归属名称 |

### 故障排除

| 问题 | 解决方案 |
|------|---------|
| 文字乱码或错误 | 将文字逐字写入提示词：`Title: "确切文字"` |
| 风格不一致 | 确保第 1 页成功生成 — 它是所有其他页的参考 |
| 速率限制错误 | 增加配置中的 `throttle_ms`（尝试 `4000`） |
| 图片空白/无响应 | 重试 — 生成器会自动重试失败的幻灯片 |
| API 密钥错误 | 在 [aistudio.google.com](https://aistudio.google.com) 验证 `GEMINI_API_KEY` 环境变量 |
